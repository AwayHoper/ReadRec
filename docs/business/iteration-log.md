# Iteration Log

## 版本记录

### v0.1.0

- 初始化项目目录结构
- 初始化业务文档、技术文档与规则文档骨架

### v0.2.0

- 新增官方词库 txt 导入脚本
- 官方词库已导入线上 PostgreSQL
- `dictionary` 模块已切换为读取真实 PostgreSQL 词库数据

### v0.3.0

- `auth / study-plan / daily-session / learning / wrong-book` 已切换到 Prisma 持久化
- 前端词库学习链路已从 `mock-api` 切换到真实 HTTP API
- 学习计划、每日学习、生词本与总结页现在基于真实 PostgreSQL 数据运行

### v0.4.0

- 首页已移除“官方词库”模块，首页职责收敛为今日学习总览
- “计划”页已改为上方词库卡片、下方学习计划的布局
- 词库选择方式已从表单选择器切换为词库卡片点击选中
- 词库卡片已支持“查看单词”入口
- 词库详情页已接入真实单词列表、分页与“已学习 / 未学习”状态展示

### v0.5.0

- 学习计划模块已从原始数字输入升级为“比例调节器 + 方案卡片 + 动态预测”
- 比例固定为 `1:1 / 1:2 / 1:3 / 1:4`
- 新词方案固定为 `5 / 10 / 20 / 40`
- 计划页会基于当前词库总词数与已学词数实时显示预计完成日期
- 切换词库时会恢复该词库对应的已保存计划方案；无计划时回退到 `1:1` 与 `10`
- 每日抽词逻辑已改为按总量和比例拆分新词 / 复习词
- 生词本中的词会优先参与复习词抽取
- 当日学习词会在写入 session 前乱序，避免字母顺序聚集

### v0.6.0

- 首页已升级为学习中枢，集中展示今日学习状态、掌握度、打卡日历与鼓励文案
- 后端新增 `GET /dashboard/home` 聚合接口，首页不再自行拼接多路旧接口
- 同一天现在支持多批次学习，`DailySession` 通过 `batchIndex` 区分批次
- 完成一轮后首页会继续显示“今日已完成”，并支持点击“再学一轮”开启下一批次
- 首页 CTA 已区分 `开始学习` 与 `再学一轮`：前者走开始当日学习，后者创建/进入同日下一批次
- 首页恢复学习时会根据当前 session 状态路由到一轮、二轮或三轮，而不会误回到第一轮
- 首页日期展示已改为本地安全解析，避免 UTC 日期字符串在部分时区显示前一天
